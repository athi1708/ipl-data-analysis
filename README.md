**🏏 IPL Insights: Data Analysis using Power BI**

**Project Overview**

This project analyzes data from the Indian Premier League using Power BI.
The goal is to extract meaningful insights about team performance, player statistics, and match outcomes through interactive dashboards.


**Objectives**
 
Analyze team performance across seasons
Identify top players (runs, wickets)
Study match-winning patterns
Visualize trends using dashboards

**Tools & Technologies**
 
Microsoft Power BI Desktop
Power Query (Data Cleaning)
DAX (Calculations)

**Dashboard Features**

Team-wise performance analysis
Top batsmen and bowlers
Toss vs match result insights
Season-wise trends
Interactive filters (slicers)



**Project Files**

IPL.pbix → Power BI dashboard file

IPL DATA(ZIP)



 **How to Use**
 
Download the .pbix file
Open in Power BI Desktop
Explore dashboards using filters


 **Future Improvements**
 
Add prediction using Machine Learning
Real-time data integration
Advanced analytics


**Conclusion**

This project demonstrates how data visualization can be used to analyze sports data and generate actionable insights.
